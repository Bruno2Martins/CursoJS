Os cinco principais metodos de selecionar elementos DOM dentro do JS

getElementsByTagName
Seleciona tags html como body, h1, p...

getElementsByName
Seleciona tags com Name

getElementsById
Seleciona tags com Id

getElementsByClassName
Seleciona tags com Classe

querySelector
Escolhe a tag que que quer selecionar tanto com id ou classe
